# 👁️ Vision Intelligence Pro

An advanced, real-time computer vision suite that brings state-of-the-art AI models to a user-friendly dashboard. This project integrates **YOLOv8** for object detection and **MediaPipe** for high-fidelity human pose and facial analysis.

---

### ✨ Features
- **Real-Time Object Detection**: High-speed detection of 80+ object classes using Ultralytics YOLOv8.
- **Human Pose Estimation**: Professional-grade skeletal tracking for motion analysis.
- **Facial Intelligence**: Real-time detection of facial landmarks and emotion estimation.
- **Interactive Dashboard**: Control detection thresholds and switch between modes instantly via Streamlit.
- **Support for Image \u0026 Video**: Process static images or real-time webcam/video streams.

---

### 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/QuangHoangDinh/Vision-Intelligence-Pro.git
   cd Vision-Intelligence-Pro
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the Dashboard**
   ```bash
   streamlit run app.py
   ```

---

### 🛠️ Technology Stack
- **AI Framework**: Ultralytics YOLOv8
- **Tracking**: MediaPipe
- **Processing**: OpenCV, NumPy
- **Dashboard**: Streamlit
- **Visualization**: Matplotlib / PIL
